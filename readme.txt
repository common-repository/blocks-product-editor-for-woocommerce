Blocks Product Editor for WooCommerce plugin enables the Gutenberg Editor for WooCommerce products.

== Description ==

Blocks Product Editor for WooCommerce plugin enables the Gutenberg Editor for [WooCommerce](https://woocommerce.com/?aff=12943&cid=17113767) products.

https://youtu.be/aCbQTk5AS0E

== FREE Features ==

* Enable or disable the Gutenberg Editor for WooCommerce products

== PRO Features ==

The [PRO](https://ethereumico.io/product/gutenberg-woocommerce-product-editor-plugin/) version includes all FREE features, plus:

* The Catalog visibility metabox
* Enable tags and categories taxonomies in REST
